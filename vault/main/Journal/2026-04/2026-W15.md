## What you'll find here
Semana dominada por infraestrutura OpenClaw e consolidação do vault. O problema central foi o vazamento de thinking tokens do GLM 5.1 — rastreado até a falta de `params.thinking: "off"` no payload da Z.ai API, corrigido em quatro camadas (agent defaults, global baseline, 21 session spawns e model catalog). O OC foi atualizado para 2026.4.5 e ajustado com limites conservadores para o plano Coding Plan. Dois novos agentes foram criados — parmeirense (🐷) e crypto-bro (💸) — com vault scaffold completo. O pipeline palmeiras-feed foi refatorado para priorizar imagens editoriais reais e excluir notícias sem data confirmada.

## Goals
- Corrigir vazamento de thinking tokens no GLM 5.1
- Padronizar vault-style architecture para instructions do OC
- Criar agentes parmeirense e crypto-bro no vault
- Refatorar pipeline palmeiras-feed com prioridade a imagens reais
- Importar API keys para vault/.env

## Decisions
- `params.thinking: "off"` aplicado em todos os modelos GLM com thinking habilitado por padrão
- `subagents.maxConcurrent: 2` como limite conservador para Z.ai (1 req in-flight simultâneo)
- Vault como padrão obrigatório para criação/edição de instructions no OC, documentado em `_vault-rules.md`
- Imagens do Palmeiras: apenas fontes editoriais reais — geração por IA (Gemini/Nano Banana) removida
- Notícias: apenas com `date_confirmed: true` — GE Globo e Lance removidos por ausência de data

## Progress
- GLM thinking leak corrigido via `thinkingDefault: "off"`, `params.thinking: "off"` no model catalog e patches nos 21 `sessions_spawn` em 5 arquivos de instructions
- OC atualizado de 2026.3.22 para 2026.4.5 e validado após reinício do gateway
- Agentes parmeirense e crypto-bro criados com vault scaffold completo e index Agents.md atualizado
- Pipeline palmeiras-feed refatorado: `fetch-images.md` com cascata de prioridades, `select-images.md` com scoring sem IA, `fetch-web.md` e `select-topics.md` com filtro de data
- 15 API keys importadas para vault/.env (Notion DBs, Z.AI, Gemini, X/Twitter, SmartThings, TMDB, Jellyfin, Finnhub)
- `_vault-rules.md` criado e referenciado em `AGENTS.md` como padrão obrigatório

## Next Week

- Monitorar logs da Z.ai para confirmar que `params.thinking: "off"` chega no payload
- Investigar changelog das 7 releases do OC 2026.3.23→2026.4.5 (conteúdo ainda desconhecido)
