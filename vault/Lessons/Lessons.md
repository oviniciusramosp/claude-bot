# Lessons

Compound-engineering knowledge base. Every routine/pipeline failure auto-drafts a lesson file here (`draft-YYYY-MM-DD-{name}.md`) with the error summary and TODO sections for Fix and Detect. The user completes the draft by filling in the fix and detection criteria, then renames it to drop the `draft-` prefix.

Manual lessons can be added via `/lesson <text>` on Telegram — they land as `manual-YYYY-MM-DD-HHMM.md`.

Consult this folder before starting tasks that resemble previous failures so you don't repeat the same mistake.
